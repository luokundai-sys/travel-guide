# 行囊 · 家庭旅游攻略

家庭共享的旅游攻略 + 行程规划 Web App。一份共享攻略库、共享行程，**全家实时同步**；部署成一个链接发给家人即可用。

- **共享**：攻略库、行程存 Supabase，谁加/改全家几秒内看到
- **个人**：收藏（♥）存各自浏览器
- **无登录**：拿到链接就能看和改（链接即进）
- **不接 AI / 零月费**：Supabase 免费档 + GitHub Pages 免费

## 上线给家人

看 **`SETUP.md`** —— 4 步，你只需建个 Supabase 项目并填 `config.js` 两个值，部署我帮你跑。

## 本地跑（开发/预览）

```bash
cd travel-guide
python3 -m http.server 8777
# 打开 http://localhost:8777/
```
没填 `config.js` 时顶部会提示"还没接后端"；填好后即连上共享数据。

## 功能

- **攻略库**：浏览/搜索/标签筛选/收藏；**「+ 新建攻略」**自己加，**「编辑」/「🗑」**改删（改动全家共享）
- **我的行程**：新建行程 → 每天一列，景点卡片拖动排序、跨天移动，填时间/备注，「+ 自定义」手加；攻略里「+」或「整份加入行程」一键带入

## 文件

| 文件 | 作用 |
|------|------|
| `index.html` | 页面结构 |
| `styles.css` | 样式 |
| `config.js` | **你填**：Supabase URL + anon key |
| `schema.sql` | Supabase 建表脚本（控制台 SQL Editor 跑一次） |
| `db.js` | 数据层（Supabase CRUD + 实时同步） |
| `seed.js` | 首次为空时的种子攻略（4 个目的地） |
| `app.js` | 全部交互逻辑 |
| `SETUP.md` | 上线分步清单 |
